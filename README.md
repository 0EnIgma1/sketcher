# Sketcher

- A program to convert any image to various sketches and filters using python.

- Available Filters
  >- Pencil Sketch
  >- Watercolor
  >- Grayscale
  >- Negative
  >- Cartoon
  >- Dynamic Canny Edge detection

- Developed in python using :-

  >- openCV
  >- Gaussian Blur
